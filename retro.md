# 🧠 Retrospectiva del Proyecto

## 📌 Descripción
Desarrollo de un ecommerce full stack con Node.js, Express, MySQL y un dashboard en React consumiendo una API propia.

---

## 🚀 Lo que logré

- Implementar registro y login de usuarios con encriptación de contraseñas
- Crear una API REST para productos y usuarios
- Conectar el backend con MySQL usando Sequelize
- Desarrollar un dashboard en React consumiendo la API
- Mostrar métricas como cantidad de productos, usuarios y categorías

---

## 🧩 Dificultades encontradas

- Problemas con Sequelize (timestamps, relaciones y queries)
- Errores en login (comparación de contraseñas con bcrypt)
- Configuración de rutas y controllers
- Conflictos de puertos entre backend y React
- Problemas de CORS al conectar frontend con backend
- Manejo de estados en React y renderizado inicial vacío

---

## 🛠 Cómo los resolví

- Revisando la configuración de modelos y base de datos
- Debuggeando con console.log para entender errores
- Separando correctamente frontend y backend
- Configurando CORS en el servidor
- Corrigiendo imports y estructura de componentes en React

---

## 📚 Aprendizajes

- Entendí cómo funciona una arquitectura full stack real
- Aprendí a consumir APIs desde React
- Mejoré en debugging y lectura de errores
- Comprendí mejor el flujo de autenticación
- Aprendí a estructurar un proyecto profesional

---

## 🔮 Mejoras futuras

- Implementar roles de usuario (admin)
- Agregar autenticación con JWT
- Mejorar el diseño del dashboard
- Deployar la aplicación online
- Agregar filtros y búsquedas de productos

---

## 💬 Conclusión

Este proyecto me permitió integrar frontend y backend en una aplicación completa, enfrentando problemas reales y resolviéndolos de forma práctica.
